# UI: 12_main_menu_interactive

## Components
- **ASCII Splash Screen:** "INFINITE LOOP" logo.
- **Menu List:** Selectable options (New Game, Load, Marketplace).
- **Selector UI:** For choosing worker profiles and mission goals.

## Interaction
- Arrow keys or Vim keys (`j`/`k`) to navigate options.
- `[ENTER]` to select.
