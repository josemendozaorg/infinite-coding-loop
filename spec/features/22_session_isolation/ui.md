# UI: 22_session_isolation

## Components
- **Session Picker:** List of saved sessions in "Load Game" menu.
- **Session Header:** Display current session name in dashboard header.
- **New Session Flow:** Wizard creates fresh session with unique ID.

## Interaction
- "New Game" always creates a new isolated session.
- "Load Game" shows list of previous sessions with goal and timestamp.
- Selecting a session restores its full state.
