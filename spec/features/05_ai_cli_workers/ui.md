# UI: 05_ai_cli_workers

## Visual Components

### 1. Debug Inspector
- **Raw Prompt Viewer**: See exactly what was sent to the LLM.
- **Raw Response Viewer**: See the raw JSON/Markdown response.
- **Token Usage**: Visual meter for context window usage (e.g., 80% of 128k tokens).

### 2. Manual Override
- Input field to manually inject a response (for testing/mocking).
